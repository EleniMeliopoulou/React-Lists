# Healthy Food/Snacks List

The lists were built using React. One list contains healthy foods, while the other list contains snacks. The user can switch between the two lists by clicking a button.

---

## Technologies Used
* **Front-End:**
    * React with Vite 
    * CSS 

---

## Getting Started

Follow the steps below to set up and run the application locally.

Clone the Repository

```bash
git clone https://github.com/EleniMeliopoulou/React-Lists.git
cd React-Lists
```

Install Dependencies

```bash
npm install
```

Start the Front-End
```bash
npm run dev
```
The application will be available at:

http://localhost:5173
